# Comparison-of-Programming-Languages
This repository contains a comparative study of fundamental programming language structures implemented across 7 different languages. This project was developed as part of the CS 315 programming languages.
Overview: The goal of this study is to analyze how different programming languages handle syntax and semantics. By implementing the same logic in multiple languages, I explored the trade-offs between readability, safety and performance.

  1. Selection statements & nested selectors: explores the design choices for conditional logic:
        Control expressions: Requirements for boolean vs. truthy values like  Rust's strictness vs. JavaScript's flexibility.
        Nesting logic: Implementation of nested if&else structures to observe how different languages handle the dangling else ambiguity.

  2. Associative Arrays: a deep dive into built-in data structures like maps, hashes, dictionaries:
        Implementation of CRUD operations.
        Use of higher order functions for iteration and searching.

Languages includes:
Rust, Go, Dart, Kotlin, Javaccript, Python, Ruby
